Remarks

The connection string is defined in the standard .NET <connectionStrings> element within the app.config file however the format is specific to CiteSet.

The connection string value is in the form of a Uri with a custom scheme: db4o://. The Uri format simply makes it easier to parse values from the string for use in configuring the CiteSet.Database.DatabaseServer class.
